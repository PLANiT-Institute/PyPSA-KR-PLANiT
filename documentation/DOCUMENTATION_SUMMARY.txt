MAINTENANCE NOTES
================================================================================

When updating code:
1. Update function docstrings in source files
2. Update corresponding sections in 02_LIBRARY_MODULES.md
3. Update process flow if workflow changes (03_PROCESS_FLOW.md)
4. Update configuration guide if config changes (04_CONFIGURATION_GUIDE.md)
5. Update examples in 01_MAIN_SCRIPTS.md if needed
6. Update README.md if major changes affect navigation

Documentation Standards:
- Clear hierarchy (##, ###, ####)
- Concrete examples with code
- Cross-references between documents
- Complete parameter documentation
- Tested code examples
